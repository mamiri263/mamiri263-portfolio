# To learn web development you need to know

## Html

To make a simple website you must have basic knowledge of html. HTML is the standard markup language for Web pages.

html files have the extention of .html

To learn html I suggest you take a look at places like:

- [Freecodecamp](https://www.freecodecamp.org/)
- [w3schools](https://www.w3schools.com/)
- [codecademy](https://www.codecademy.com/)

## css 

css is a declarative language for styling web pages or to make it simple styling your webpage.

css files have the extention of .css

To learn css I suggest you take a look at places like:

- [web.dev](https://web.dev/learn/css/)
- [codecademy](https://www.codecademy.com/learn/learn-css)
- youtube videos about css and people reviewing css code.

## javascript

You don't need much javascript if you are just starting but it helps you do pretty amazing things and create functions on your website.

javascript files have the extention of .js

To learn javascript you can visit websites like:

- [learnjavascript](https://learnjavascript.online/)
- [learn-js](https://www.learn-js.org/)
- [programming with mosh](https://www.youtube.com/watch?v=upDLs1sn7g4&list=PLTjRvDozrdlxEIuOBZkMAK5uiqp8rHUax)
